POLITIQUE DE CONFIDENTIALITÉ

Application Mots Secrets — En vigueur depuis le 5 mars 2026

La présente politique de confidentialité s'applique à l'application mobile Mots Secrets, développée par Rallow. En utilisant cette application, vous acceptez les pratiques décrites dans ce document.

1. Données collectées

L'Application collecte les données suivantes via Google AdMob :

  • Identifiant publicitaire de l'appareil (GAID sur Android, IDFA sur iOS)
  • Adresse IP de l'appareil
  • Informations sur l'appareil (modèle, système d'exploitation, version)
  • Données d'utilisation de l'application (pages visitées, temps passé)
  • Interactions avec les publicités affichées
  • Données de comportement pour la personnalisation des publicités

L'Application ne collecte PAS de données de localisation précise, ni de contacts, ni de données sensibles personnelles.

2. Publicités personnalisées (AdMob)

L'Application utilise Google AdMob pour afficher des publicités personnalisées basées sur vos centres d'intérêt. Lors de votre première utilisation, un bandeau de consentement vous sera présenté conformément au RGPD.

Si vous acceptez :

  • Des publicités personnalisées seront affichées selon vos intérêts
  • Google et ses partenaires collectent des données pour cibler les annonces

Si vous refusez :

  • Des publicités non personnalisées seront affichées
  • Moins de données sont collectées
  • Vous pouvez modifier votre choix à tout moment

Pour plus d'informations : Politique de confidentialité Google

3. Base légale du traitement (RGPD)

  • Consentement : pour les publicités personnalisées (révocable à tout moment)
  • Intérêt légitime : pour le bon fonctionnement de l'application

4. Vos droits

Conformément au RGPD, vous disposez des droits suivants :

  • Droit d'accès : obtenir une copie de vos données personnelles
  • Droit de rectification : corriger des données inexactes
  • Droit à l'effacement : demander la suppression de vos données
  • Droit d'opposition : vous opposer au traitement de vos données
  • Droit à la portabilité : recevoir vos données dans un format structuré
  • Droit de retrait du consentement : modifier votre choix aux publicités

Pour exercer ces droits : washdenzel90@gmail.com

5. Partage des données

  • Google LLC (AdMob) — pour la diffusion des publicités
  • Partenaires publicitaires de Google — uniquement avec votre consentement
  • Autorités légales — si la loi l'exige

Nous ne vendons jamais vos données personnelles à des tiers.

6. Conservation des données

Les données sont conservées tant que l'application est installée. En désinstallant l'application, la collecte cesse. Pour demander la suppression : washdenzel90@gmail.com

7. Mineurs

L'Application n'est pas destinée aux enfants de moins de 13 ans. Si vous êtes parent et constatez que votre enfant a utilisé l'application, contactez-nous à washdenzel90@gmail.com.

8. Sécurité

Nous mettons en œuvre des mesures techniques appropriées pour protéger vos données contre tout accès non autorisé.

9. Modifications

Cette politique peut être mise à jour à tout moment. Nous vous informerons de tout changement via l'application ou par email.

11. Révoquer votre consentement

Vous pouvez modifier ou révoquer votre consentement aux publicités personnalisées à tout moment directement dans l'application via le bouton "GÉRER MES DONNÉES" sur l'écran principal de l'application.
